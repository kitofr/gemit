REAME
=====

Installation
------------
gem install gemit

Usage
-----
gemit [gem name]


    $> gemit new_gem
    mkdir -p new_gem/bin
    mkdir -p new_gem/lib
    mkdir -p new_gem/lib/new_gem
    created new_gem/new_gem.gemspec
    created new_gem/README.md
    created new_gem/lib/new_gem/version.rb
    created new_gem/lib/new_gem.rb
    created new_gem/bin/new_gem
